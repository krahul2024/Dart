| Command       | Description                  |
| ------------- | ---------------------------- |
| `add <task>`  | Add a new task               |
| `list`        | Show all tasks               |
| `done <id>`   | Mark a task as completed     |
| `undo <id>`   | Mark a task as not completed |
| `delete <id>` | Delete a task                |
| `clear`       | Delete all tasks             |
| `help`        | Show usage/help info         |
| `exit`        | Exit the app (if REPL style) |
